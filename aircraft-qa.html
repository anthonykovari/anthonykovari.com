Aircraft Appearance Assessment Tool

React · Python · TensorFlow · United Airlines · 2023

Built for United Airlines QA reviewers to assess aircraft appearance from social media posts. A multi-model ML pipeline automatically ingests posts, runs them through a garbage classifier, interior/exterior classifier, paint quality scorer (0–10), and sentiment analyzer before surfacing results in a React dashboard for triage and action.
