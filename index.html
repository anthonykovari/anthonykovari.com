🗺️ ML Pipeline — Chicagoland Land Use Change

Full-stack production ML platform monitoring land-use change across 13,000 km² of the Chicago metro using real Sentinel-2 satellite imagery (2019–2026). ResNet-18 fine-tuned on EuroSAT achieves 97–99% per-class accuracy and exports to ONNX. Imagery is ingested from the Copernicus CDSE API; models are versioned in MLflow; FastAPI serves predictions on AWS EKS with full Terraform infra-as-code, GitHub Actions CI/CD, and Prometheus/Grafana observability.

PyTorch ONNX Apache Airflow MLflow AWS EKS Terraform Prometheus GitHub Actions
